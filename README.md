# SMILE DETECTION USING COMPUTER VISION

## OVERVIEW
- Simple project to understand **Computer Vision** techniques.  
- Detects and captures **smiling faces** in a video stream.  

## TECHNOLOGIES USED
- **OpenCV (cv2)** for real-time video capture and processing.  
- **CascadeClassifier** for detecting faces and smiles using Haar cascades.  

## KEY FEATURES
- Real-time face and smile detection.  
- Highlights detected smiles with bounding boxes.  
- Demonstrates practical use of pre-trained Haar cascade models.
